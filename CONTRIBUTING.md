# Contributing to Free Motion

Thanks for contributing to Free Motion.

Free Motion is **OpenClaw → Telegram → Raspberry Pi** with **two on-device models** (vision + mission control) driving flight. Contributions that keep that story clear and safe are especially welcome.

## Ways to help
- Fix bugs in the Telegram → Pi path or model integration
- Improve the docs (OpenClaw, Pi setup, vision vs mission control)
- Add Raspberry Pi examples and flight-safety notes
- Harden or extend support for more drones or robots

## How to contribute
1. Fork the repo
2. Create a branch
3. Make your changes
4. Open a pull request
